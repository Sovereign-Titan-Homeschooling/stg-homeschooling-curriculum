# 🧪 LAB ROBOTIKA 1: KOMANDO ARMADA
**Tugas**: Menulis skrip Bash untuk mengerahkan armada robot.

## Instruksi:
Buatlah file bernama `deploy_harvester.sh` yang melakukan hal berikut:
1. Menampilkan pesan: "🏛️ STG COMMAND: Deploying 1,000 Harvester Units to Asteroid 16 Psyche..."
2. Membuat folder `telemetry_data`.
3. Membuat file `status_report.log` berisi: "All Systems Nominal. Power 100%."

## Cara Menilai:
Jalankan `./scripts/grade_week_1.sh` untuk melihat apakah logika strukturmu sudah benar.
